# 恋爱小屋 - 部署指南

## 🚀 快速部署（免费）

### 方法一：Render.com 部署（推荐，最简单）

1. 把项目文件夹上传到 GitHub 仓库
2. 打开 [render.com](https://render.com)，用 GitHub 注册
3. 点 "New Web Service"，选择你的仓库
4. 设置：
   - Build Command: `npm install`
   - Start Command: `node server.js`
5. 点 "Create Web Service"
6. 等2分钟，拿到网址如 `https://love-house.onrender.com`
7. 发给对方，搞定！

### 方法二：同一WiFi直接访问（临时）

电脑开服务器后，手机连同一WiFi，浏览器打开：
```
http://172.16.72.7:3000
```

### 方法三：Railway.app 部署

1. 打开 [railway.app](https://railway.app)，用 GitHub 注册
2. 点 "New Project" → "Deploy from GitHub repo"
3. 选择仓库，自动部署
4. 拿到网址
